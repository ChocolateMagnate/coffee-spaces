About the author

This product was originally initiated and built by Vlad Korol in 2022. With this project, he strived to make a user-friendly service for software engineers that would simplify their work and provide them with everything they need in one service. This is why the service has grown up into the following features:

For testers:
  • rich bug tracker
  • communication line with developers
  • blazing notifications that appear right when progress was made
For developers:
  1. highlighting code editor
  2. complete review of codebase
  3. simple bug review
For project managers:
  • tools for managing projects configurations, members, access and other permissions.
  • complete review of the codebase
